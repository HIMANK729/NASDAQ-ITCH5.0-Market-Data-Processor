# FPGA-NASDAQ-ITCH-5.0-Market-Data-Processor
A  Verilog implementation of a NASDAQ ITCH 5.0 market data processing pipeline. The design parses Ethernet, IPv4, and UDP packets, decodes ITCH messages, maintains a real-time order book, generates Best Bid/Offer (BBO) updates, and measures packet-to-book update latency.
